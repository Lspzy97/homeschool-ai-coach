# 情直理论 - 完整框架

## 理论来源
由心理学博主6日提出并逐步完善，基于脑科学、心理动力学、进化心理学，结合10余年情感和心理咨询案例。

## 核心概念

### 一、情与直（两种基本类型）

**情类型**：情感导向，敏感、情绪化、注重感受和情绪价值，需要被理解和共情
**直类型**：理性导向，务实、直接、注重逻辑和结果，需要被尊重和认可

### 二、男性分类维度：多偶 vs 单偶

**单偶倾向**：倾向于一对一忠诚关系，重视家庭稳定，对伴侣有保护欲
**多偶倾向**：倾向于多偶开放式关系，追逐新鲜感，不喜欢被束缚

### 三、女性分类维度：攻 vs 守

**攻型**：主动、强势、喜欢主导关系，追求掌控感
**守型**：被动、柔和、愿意配合伴侣，倾向于稳定和谐

## 八种情感类型

### 男性四种类型

| 类型 | 名称 | 特点 |
|------|------|------|
| 情×单偶 | 单偶（老实）情男 | 专一忠诚但情绪化，重视感情但不善表达 |
| 情×多偶 | 多偶（强势）情男 | 情感丰富但多偶倾向，浪漫但可能花心 |
| 直×单偶 | 单偶直男 | 专一务实理性，不善言辞但行动证明 |
| 直×多偶 | 多偶直男 | 多偶倾向但理性务实，目标导向 |

### 女性四种类型

| 类型 | 名称 | 特点 |
|------|------|------|
| 情×攻 | 攻情女 | 主动追求但情感敏感，占有欲强 |
| 情×守 | 守情女 | 被动等待但情感细腻，温柔包容 |
| 直×攻 | 公直女 | 主动直接但理性务实，性格像兄弟 |
| 直×守 | 手侄女 | 被动配合但务实乖巧，依赖伴侣 |

## 稳定关系原则：情执互配

### 核心配对公式

| 原则 | 说明 |
|------|------|
| 情执互配 | 情感型的人需要情感丰富的伴侣，理性型需要理性型的伴侣 |
| 攻对单 | 主动型女性 ↔ 单偶倾向男性 |
| 守对多 | 被动型女性 ↔ 多偶倾向男性 |

### 配对解析

- **情执互配**：情感型的人在一起能相互理解对方的需求和感受
- **攻对单**：主动女性配专一男性，形成女主主导但男方忠诚的关系
- **守对多**：被动女性配多偶男性，女方包容度能接受男方的多偶倾向

## 理论应用

### 识人辨认维度（判断对方类型）

1. **情绪反应模式**：敏感vs钝感
2. **决策方式**：情感驱动vs理性驱动
3. **关系态度**：专一vs开放
4. **行为模式**：主动vs被动
5. **表达方式**：含蓄vs直接

### 沟通适配策略

| 类型 | 适合的沟通方式 |
|------|--------------|
| 单偶直男 | 直接、务实、给予安全感 |
| 多偶直男 | 保持神秘感、保持独立 |
| 单偶情男 | 情感共鸣、表达理解、陪伴 |
| 多偶情男 | 浪漫、制造惊喜、情感丰富 |
| 攻情女 | 尊重她的主动、给予舞台 |
| 守情女 | 主动关心、温柔呵护 |
