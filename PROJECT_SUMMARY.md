# 家校AI沟通教练 - 项目概要

## 产品定位
**AI 沟通教练** —— 帮教育工作者更懂如何跟学生/家长说话

## 核心价值
- 对学生：有朋友视角，知道学生最近发生了什么
- 对家长：专业同理心，帮家长理解孩子行为背后的原因
- 工具帮你生成：家长能看懂、愿意接受的沟通文案

## 目标用户
- B端：服务于素养类教育机构
- C端：服务于独立老师、家长、学生

## 产品规划（三个阶段）

### 第一阶段：对话风格学习 ✅
- 收集聊天记录，建立个人说话风格数据库
- 学会用你的语气生成回复
- 用户网名：**朝也**

### 第二阶段：角色心理分析 ✅（精进中）
- 加入心理学知识库
- 分析学生/家长/老师不同角色的心理动机
- 识别沟通对象的特点

### 第三阶段：智能话术切换
- 根据对方性格自动适配沟通方式
- 输出对方能听进去的表达

## 技术方案

| 模块   | 技术选择     | 原因                     |
|--------|-------------|------------------------|
| 前端   | 微信小程序   | 你之前做过，有经验       |
| 后端   | 云开发       | 免费额度够用，简单        |
| AI对话 | MiniMax API | 便宜、效果好             |
| 数据库 | 微信云数据库 | 和小程序无缝结合          |

## 核心功能（输入模式）
1. 分析与一个人的聊天记录，识别对方属性
2. 老师在微信聊天框打出想回复的消息
3. 点击小按钮，算法生成更适合的文案替换
4. 用户看过后自行决定要不要发送

## 识人辨人理论体系

### 情直理论（核心C位）
基于脑科学、心理动力学、进化心理学，由心理学博主6日提出

| 男性类型 | 名称 | 特点 |
|---------|------|------|
| 情×单偶 | 单偶（老实）情男 | 专一忠诚但情绪化，重视感情但不善表达 |
| 情×多偶 | 多偶（强势）情男 | 情感丰富但多偶倾向，浪漫但可能花心 |
| 直×单偶 | 单偶直男 | 专一务实理性，不善言辞但行动证明 |
| 直×多偶 | 多偶直男 | 多偶倾向但理性务实，目标导向 |

| 女性类型 | 名称 | 特点 |
|---------|------|------|
| 情×攻 | 攻情女 | 主动追求但情感敏感，占有欲强 |
| 情×守 | 守情女 | 被动等待但情感细腻，温柔包容 |
| 直×攻 | 公直女 | 主动直接但理性务实，性格像兄弟 |
| 直×守 | 手侄女 | 被动配合但务实乖巧，依赖伴侣 |

### 心理学多维度分析（新增）

| 维度 | 理论来源 | 分析内容 |
|------|---------|---------|
| 依恋类型 | 鲍尔比 | 安全型/焦虑型/回避型/混乱型 |
| 萨提亚沟通姿态 | 维吉尼亚·萨提亚 | 讨好型/指责型/超理智型/打岔型/一致型 |
| 大五人格 | OCEAN模型 | 开放性/尽责性/外向性/宜人性/神经质 |
| 学习风格 | VARK模型 | 视觉型/听觉型/读写型/动觉型 |
| 情商五要素 | 戈尔曼 | 自我认知/自我管理/社会认知/关系管理/自我激励 |
| 认知偏差 | 认知心理学 | 基本归因错误/确认偏差/损失厌恶/近期偏差等 |
| 自我决定理论 | 德西&瑞安 | 自主需求/胜任需求/归属需求 |

### 回复质量优化
- 萨提亚一致型沟通原则：既尊重自己也尊重对方
- 识别四种不良沟通姿态，避免讨好/指责/超理智/打岔
- 像真人发微信一样自然，不AI腔
